# 《命运赌局》视觉设计方案（第一版）

## 一、整体视觉方向

游戏核心气质：

- 暗黑赌场
- 疯狂马戏团
- 心理博弈
- 赛博舞台
- 命运赌局
- 幻觉感
- 强演出
- 高对比

整体风格参考：

- Balatro
- 女神异闻录5
- Inscryption
- Library Of Ruina
- 小丑 / Joker 美学
- 赛博赌场

不要做：

- 普通仙侠UI
- 普通二次元UI
- 廉价手游风
- 高饱和页游风
- 写实3D

核心方向：

# 风格化2D
# 强UI
# 强特效
# 强情绪

# 先把“高级感”做出来。

# 十、最重要原则

这个游戏：

不是：

“传统卡牌游戏”。

而是：

# 心理赌局演出游戏。

所以：

一切视觉：

都应该强化：

- 赌博感
- 疯狂感
- 舞台感
- 压迫感
- 命运感
- 高潮感
